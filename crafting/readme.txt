**Feature**
- Tabela completa de construção do sistema de IU
- Menu fácil de usar
- dividido em categorias
- costumização
    - Adicionar uma mesa
    - Adicionar categoria
    - Adicionar itens
    - Aumente o número de objetos obtidos pela elaboração.
    - pode adicionar pontos para contar a tabela no mapa
    - Ajustado o tempo de elaboração de cada item. Pode ser multiplicado por pedaços
    - Ajustou a taxa de falha de cada item.
    - Linguagem personalizável
- Não é repetitivo e único definitivamente não é revelador.
- Atualizações grátis para toda a vida Se o vendedor tiver uma nova atualização

** Scripts necessários para trabalhar juntos
- pNotify

**วิธีติดตั้ง**
1. โยนลง resources ของ fivem
2. เพิ่ม start scotty-crafting ลงใน server.cfg
3. ใส่ license key ที่ได้จากผู้ขายใส่ลงในไฟล์ config_sv.lua
4. ปรับแต่งของที่สามารถทำได้ในไฟล์ config.lua
5. เนื่องจากรูปไอเทมในแต่ละเซิฟจะไม่เหมือนกันกรุณาอย่าลืมใส่ไอคอนลงใน html/images/items
และเพิ่มที่อยู่ลงใน _resource.lua
